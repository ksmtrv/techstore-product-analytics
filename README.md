# TechStore Product Analytics

## О проекте

TechStore — интернет-магазин электроники.

Компания заметила, что количество пользователей продолжает расти, однако прибыль увеличивается значительно медленнее ожидаемой. Для поиска причин необходимо провести анализ пользовательского поведения, продаж и ключевых продуктовых метрик.

В данном проекте я выступаю в роли **бизнес-аналитика**, задача которого — найти проблемы, сформулировать гипотезы и предложить решения на основе данных.

---

## Цель проекта

Провести комплексный анализ данных интернет-магазина и определить направления для увеличения прибыли.

---

## Задачи

- спроектировать структуру данных;
- подготовить тестовый датасет;
- провести анализ с помощью SQL;
- рассчитать продуктовые метрики;
- визуализировать результаты;
- подготовить рекомендации для бизнеса.
