# TechStore Product Analytics

Проект по анализу данных интернет-магазина электроники с использованием **Python**, **SQL** и **Jupyter Notebook**.

## Что сделано

- генерация тестовых данных;
- исследовательский анализ данных (EDA);
- анализ пользователей, товаров и продаж;
- построение визуализаций;
- SQL-запросы для анализа данных.

## Стек

- Python
- Pandas
- Matplotlib
- PostgreSQL
- SQL
- Jupyter Notebook
- Faker
- Git

## Структура проекта

```
data/          # Данные
docs/          # Документация
python/        # Генерация данных и анализ
sql/           # SQL-скрипты
```

## Запуск

```bash
pip install -r requirements.txt
python python/generate_data.py
jupyter notebook
```

Открыть:

```
python/analysis.ipynb
```

## Результат

В ходе проекта выполнен анализ данных интернет-магазина, рассчитаны основные бизнес-показатели и построены визуализации для исследования пользователей, товаров и продаж.
