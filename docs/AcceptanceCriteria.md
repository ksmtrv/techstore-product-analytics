# Acceptance Criteria

## Анализ пользователей

- отображается количество пользователей;
- отображается количество новых регистраций;
- доступны данные по городам.

---

## Анализ заказов

- отображается средний чек;
- отображается количество заказов;
- отображается сумма продаж.

---

## Анализ товаров

- доступны продажи по категориям;
- доступны продажи по брендам;
- доступны самые популярные товары.

---

## Анализ трафика

- отображаются устройства пользователей;
- отображаются источники трафика.

---

## Итоговый отчет

После завершения анализа должны быть сформированы рекомендации для бизнеса.
