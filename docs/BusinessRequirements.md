# Бизнес-требования

## Описание проекта

TechStore — интернет-магазин электроники.

За последние полгода количество зарегистрированных пользователей выросло более чем на 30%, однако темпы роста прибыли значительно снизились.

Руководство компании предполагает наличие проблем в процессе оформления заказа, работе каталога или поведении пользователей на сайте.

Для принятия дальнейших решений необходимо провести анализ данных.

---

## Бизнес-проблема

Рост количества пользователей не приводит к пропорциональному росту прибыли.

---

## Цель

Найти причины снижения эффективности продаж и подготовить предложения по развитию продукта.

---

## Основные задачи

- определить наиболее прибыльные категории товаров;
- определить категории с низкими продажами;
- проанализировать активность пользователей;
- оценить повторные покупки;
- рассчитать основные продуктовые метрики;
- подготовить рекомендации для бизнеса.

---

## Ожидаемый результат

По итогам анализа руководство должно получить:

- выявленные проблемы;
- рассчитанные метрики;
- список гипотез;
- рекомендации по развитию продукта.
